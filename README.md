# WordSpark App

WordSpark App คือเว็บแอปพลิเคชันสำหรับสุ่มคำตามหมวดหมู่ต่างๆ เพื่อช่วยกระตุ้นจินตนาการและฝึกทักษะการเล่าเรื่อง (Storytelling)

## วัตถุประสงค์ (Project Overview)
แอปพลิเคชันนี้ออกแบบมาเพื่อช่วยให้ผู้ใช้งานได้ฝึกฝนการเล่าเรื่องจากคำที่สุ่มได้ โดยเน้นความง่ายและความสนุกในการใช้งาน

## ฟีเจอร์หลัก (Key Features)
- **สุ่มคำตามหมวดหมู่:** เลือกสุ่มคำจากหมวดหมู่ที่หลากหลาย เช่น
  - สถานที่ (Places)
  - อารมณ์ (Emotions)
  - สัตว์ (Animals)
- **การใช้งานที่เรียบง่าย:** กดสุ่ม 1 ครั้ง จะแสดงผล 1 คำทันที
- **ฝึกการเล่าเรื่อง:** เมื่อได้คำที่สุ่มมาแล้ว ผู้ใช้สามารถเริ่มฝึกเล่าเรื่องตามจินตนาการ
- **สุ่มใหม่ได้ตลอด:** เมื่อเล่าเรื่องจบ สามารถกดสุ่มคำใหม่เพื่อเริ่มความท้าทายถัดไปได้ทันที

## เทคโนโลยีที่ใช้ (Tech Stack)
- **Frontend:** React 19
- **Build Tool:** Vite
- **Styling:** CSS

## การเริ่มต้นใช้งาน (Getting Started)

1. ติดตั้ง Dependencies:
   ```bash
   npm install
   ```

2. รันแอปพลิเคชัน (Development Mode):
   ```bash
   npm run dev
   ```

3. สร้างไฟล์สำหรับ Production:
   ```bash
   npm run build
   ```
