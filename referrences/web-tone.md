# **UX/UI Design Specification: Minimalist Random Word Generator**

**Version:** 1.0

**Role:** Senior UX/UI Product Designer

**Focus:** Fluidity, Minimalism, and High Performance

## **1\. Design Goals Table**

| Element | Functionality | UX/UI Strategy |
| :---- | :---- | :---- |
| **Hero Display** | แสดงคำหลักที่สุ่มได้ | **Visual Focus 70%:** ใช้ตัวหนา (Bold) ขนาดใหญ่ กึ่งกลางหน้าจอ |
| **Interaction Core** | ปุ่ม Generate / สุ่มคำ | **Center of Action:** ปุ่มขนาดใหญ่ กดง่าย รองรับทั้ง Touch และ Click |
| **Content Filters** | เลือกหมวดหมู่คำ | **Contextual Control:** ใช้ Pill-shaped chips เพื่อความประหยัดพื้นที่ |
| **History System** | ดูย่อหน้า/คำที่สุ่มไปแล้ว | **Seamless Access:** ออกแบบเป็น Overlay หรือ Slide-out เพื่อไม่ให้รบกวนหน้าหลัก |
| **Copy Feature** | คัดลอกคำไปใช้งาน | **Micro-feedback:** มีสถานะ "Copied\!" แจ้งเตือนทันทีหลังกด |

## **2\. Design System & Visual Identity**

### **2.1 Color Palette (WCAG 2.1 Compliant)**

* **Light Mode:**  
  * Background: \#F8F9FA (Off-white)  
  * Primary Text: \#1E293B (Dark Slate)  
  * Accent: \#6366F1 (Indigo)  
* **Dark Mode:**  
  * Background: \#0F172A (Deep Blue)  
  * Primary Text: \#F1F5F9 (Light Gray)  
  * Accent: \#818CF8 (Soft Indigo)

### **2.2 Typography**

* **Primary Font:** Inter หรือ Plus Jakarta Sans (Sans-serif)  
* **Word Display:** 6rem (Desktop) / 3.5rem (Mobile) \- **Font Weight: 700**  
* **Interface Text:** 1rem \- **Font Weight: 400/500**
